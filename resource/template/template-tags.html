模板中支持的 Go 符号

  • ${"string"} // 一般 string
  • ${`raw string`} // 原始 string
  • ${'c'} // byte
  • ${print nil} // nil 也被支持

模板中的判断

${if .Name} hello ${.Name} ${else} default name ${end}

判断嵌套示例

${if .User}

用户名: ${.User.Name}

${if .User.Admin}

角色: 管理员

${else}

角色: 普通用户

${end} ${else}

用户未登录

${end}

用户列表

    ${range .users}
  • 名字: ${.Name}, 年龄: ${.Age} ${if .Admin} (管理员) ${end}
    • ${else}
  • 没有用户
    • ${end}

用户信息

${with .user}

名字: ${.Name}

年龄: ${.Age}

${if .Admin}

角色: 管理员

${else}

角色: 普通用户

${end} ${else}

用户不存在

${end}
